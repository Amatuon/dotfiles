# Dotfiles

My personal dotfiles repository for managing configurations across machines.

Includes Aether theme 'Scopenear' with custom wallpaper and configs for Hyprland, Alacritty, etc.

## Installation

### Automated (Recommended)
1. Clone the repository:
   ```
   git clone https://github.com/Amatuon/dotfiles.git ~/dotfiles
   ```
2. Run the install script:
   ```
   cd ~/dotfiles
   chmod +x install.sh
   ./install.sh
   ```
